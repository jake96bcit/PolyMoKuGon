How to play this game?

1. This game is made based on "tic tac toe" logic but you have to hit FIVE in a row in order to win the game!

2. Since the cell is in hexagon so vertical line is not a win line.

3. The game board will renew when a player win the game.

4. Last thing is HAVE FUN and ENJOY THE GAME!
